Follow these simple steps to set up your pharmacy database

1 Open Supabase Dashboard

Go to your Supabase project dashboard:

https://app.supabase.com/project/YOUR_PROJECT_ID

Then navigate to: SQL Editor (in the left sidebar)

2 Open the Schema File

In your project folder, find and open:

hospital-crm-pro-new/PHARMACY_MODULE_SCHEMA.sql

Open it in any text editor (VS Code, Notepad++, etc.)

3 Copy & Paste the SQL
  1. Select ALL content from the PHARMACY_MODULE_SCHEMA.sql file (Ctrl+A)
  2. Copy it (Ctrl+C)
  3. Go to Supabase SQL Editor
  4. Click "+ New Query"
  5. Paste the entire content (Ctrl+V)
4 Run the Schema

In Supabase SQL Editor:

  1. Click the green "Run" button (or press Ctrl+Enter)
  2. Wait for 1-2 minutes for completion
  3. You should see "Success" message
5 Verify Setup

Run this query in Supabase to verify:

SELECT table_name FROM information_schema.tables WHERE table_schema = 'public' AND table_name LIKE 'pharmacy_%';

You should see 15+ pharmacy tables listed!
